# Unifacisa - Receita Que Dói Menos 📲 

Repositório para alocar a aplicação mobile desenvolvida como projeto de avaliação da competência <strong>Desenvolver Aplicativos Para Dispositivos Móveis e IOT</strong> da Unifacisa.

## Getting Started ▶️

O colaborador deverá possuir credenciais de acesso ao repositório e estar associado ao grupo do projeto.

Todos os pré-requisitos necessários para execução do projeto deverão ser devidamente instalados na máquina local do desenvolvedor.

### Pré-requisitos ⚙️

- [Repositório](https://github.com/darllinsonazvd/receita-que-doi-menos-mobile)
- [Node.js](https://nodejs.org/en)
- [Node Version Manager (Opcional)](https://github.com/nvm-sh/nvm)

### Instalando as Dependências 💾

```bash
npm install
```

### Inicializando a Aplicação ✅

```bash
npx expo start
```
